# MIC Made-in-China Bulk Fill

一个 Hermes Skill + Chrome Extension 组合工具，用于批量填充 [Made-in-China.com](https://www.made-in-china.com) 产品发布页面。

---

## 两部分组成

| 组件 | 路径 | 作用 |
|------|------|------|
| **Hermes Skill** | `hermes-skill/` | 根据产品资料生成结构化粘贴文本 |
| **Chrome 插件** | `chrome-extension/` | 将文本按板块自动填入 MIC 产品发布页 |

---

## 使用流程

1. **生成文本**：让 Hermes agent 使用 `mic-made-in-china-fill` skill，根据产品资料生成一段“粘贴文本”。
2. **展开页面**：打开 Made-in-China 产品填写页面，先点击需要展开/添加的输入项。
3. **打开插件**：打开 Chrome 插件 `Made-in-China Bulk Fill`。
4. **粘贴文本**：把 Hermes 生成的文本粘贴进插件输入框。
5. **预览确认**：先点“预览”，确认字段匹配无误。
6. **执行填充**：没问题再点“填充”。

## 关键约定

- **板块化填充**：插件按板块 DOM 填充，不再全页面猜字段，避免错填。
- **规格管理**：只放规格组，例如 `规格名称：Voltage`、`规格值：12V; 24V`。
- **FOB 价格**：会根据字段自动选择一口价、阶梯价或区间价。
- **电压**：标准属性必填，只能填一个值（如 `12V`）；多电压版本放到规格管理。
- **IP Rating**：可作为自定义属性，和标准属性 `防护等级` 不冲突。
- **样品描述**：控制在 45 字以内。
- **空字段**：不要输出，留空即可。

## 安装 Chrome 插件

1. 打开 Chrome → 扩展程序 → 管理扩展程序 → 开启“开发者模式”。
2. 点击“加载已解压的扩展程序”。
3. 选择 `chrome-extension/` 文件夹。

---

## 安装 Hermes Skill

将 `hermes-skill/` 文件夹复制到 `~/.hermes/skills/mic-made-in-china-fill/`，或按 Hermes Agent 的 skill 加载方式配置。
